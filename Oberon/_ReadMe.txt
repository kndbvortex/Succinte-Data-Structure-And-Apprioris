PRÉREQUIS

Pour pouvoir exécuter notre algorithme il faudrait avoir installer le compilateur oberon!

        *********Vérification de l'installation : ********** 

    Ouvrez une console et taper la commande obnc -V . si vous obtenez comme résultat commande introuvable alors vous devez installer gcc . 
Dépendant de votre système d'exploitation suivez les étapes suivantes : 
    
    Sur Linux:
    	allez dans le répertoire compilateur et suivez les instructions .
        Ou encore aller à l'adresse https://github.com/Spivoxity/obc-3 et suivrez les instructions présent dans le readme.Md

    Sur Windows :
    nous allons installer le compilateur oberon en nous rendant à l'adresse https://github.com/Spivoxity/obc-3/releases/download/rel-3.1.2/obc-win64-3-1-2.exe  puis télécharger la version compatible à notre syqtème d'exploitation.Téléchargez puis lancez l'exécutable .


        ******* Compilation et Exécution de l'algorithme *************

Maintenant que nous disposons de notre compilateur nous aller dans le répertoire contenant notre fichier à exécuter :
Nous allons donc ouvrir un terminal
    Sur Windows 
        il suffit de cliquer dans la barre d'adresse, d'effacer son contenu, de taper cmd puis Entrer !
    Sur Linux :
        faite un clic droit puis cliquer sur ouvrir dans un terminal / Konsole !

### Exécution des apprioris
Dans la fenêtre noire (la console) qui s'affiche , entrer les commandes suivantes :
    - obnc Main.Mod 
    Valider en appuyant sur la touche ENTRÉE
    - ./Main
    Valider en appuyant sur la touche ENTRÉE
