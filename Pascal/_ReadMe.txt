PRÉREQUIS

Pour pouvoir exécuter notre algorithme il faudrait avoir installer le compilateur freepascal!

        *********Vérification de l'installation : ********** 

    Ouvrez une console et taper la commande fpc -V . si vous obtenez comme résultat commande introuvable alors vous devez installer gcc . 
Dépendant de votre système d'exploitation suivez les étapes suivantes : 
    
    Sur Linux:
    	Ouvrez un terminal et entrez la commande 
    	sudo apt install lazarus

    Sur Windows :
        Ici nous ne disposons pas de ce compilateur il va falloir l'installer ! Forte heureusement cela ne nous prendra que quelques minutes
    nous allons installer freepascal en nous rendant à l'adresse https://www.freepascal.org/download.html puis télécharger la version compatible à notre syqtème d'exploitation.Téléchargez puis lancez l'exécutable .


        ******* Compilation et Exécution de l'algorithme *************

Maintenant que nous disposons de notre compilateur nous aller dans le répertoire contenant notre fichier à exécuter :
    Sur Windows 
        il suffit de cliquer dans la barre d'adresse, d'effacer son contenu, de taper cmd puis Entrer !
    Sur Linux :
        faite un clic droit puis cliquer sur ouvrir dans un terminal / Konsole !

Dans la fenêtre noire (la console) qui s'affiche , deplacez vous dans le dossier pascal et  entrer les commandes suivantes :
    - fpc MAIN.pas 
    Valider en appuyant sur la touche ENTRÉE
    - ./MAIN
    Valider en appuyant sur la touche ENTRÉE

Sous vos yeux s'affiche l'exécution de notre programme
